# 点名

## 确认点名

``` js
crsdk.confirmRollCall().then((res) => {
    // res 响应值看下表
});
```

| 参数名       | 类型       | 描述                                                  |
| ------------ | ------------| ----------------------------------------------------- |
| code   | Number |响应码 |
| message   | String |响应提示 |

## 订阅开始点名

``` js
crsdk.on("rollCallStarted", (res) => {
    // res 响应值看下表
});
```

| 参数名       | 类型       | 描述                                                  |
| ------------ | ------------ | ----------------------------------------------------- |
| code   | Number |响应码 |
| message   | String |响应提示 |

## 订阅结束点名

``` js
crsdk.on("rollCallStarted", (res) => {
    // res 响应值看下表
});
```

| 参数名       | 类型       | 描述                                                  |
| ------------ | ------------ | ----------------------------------------------------- |
| code   | Number |响应码 |
| message   | String |响应提示 |
